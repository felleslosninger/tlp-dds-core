# Web components

[![](https://img.shields.io/npm/v/@digdir/ds-web-react?label=@digdir/ds-web-react&color=0051be)](https://www.npmjs.com/package/@digdir/ds-web-react)

På denne siden finner du instrukter til hvordan du kan ta i bruk web komponentene.

## Kom i gang

#### NPM

[![](https://img.shields.io/npm/dm/@digdir/ds-web-react.svg)](https://www.npmjs.com/package/@digdir/ds-web-react)

Bruk av web components gjennom NPM (Node Package Manager).

#### 1. Installer pakken i kommandolinjen ved hjelp av NPM eller Yarn:

##### NPM
`npm install @digdir/ds-web-react`
##### Yarn
`yarn add @digdir/ds-web-react`

#### 2. Installer avhengigheter:

##### NPM
`npm install @digdir/ds-tokens react-bootstrap styled-components`
##### Yarn
`yarn add @digdir/ds-tokens react-bootstrap styled-components`

#### 3.Importer komponentene inn og ta de i bruk på denne måten:

##### JavaScript / Typescript

<CodeSnippet children='// Importer
import { TeaserCard } from "@digdir/ds-web-react"
// Bruk komponenten
<TeaserCard title="Here is a title" desc="Here is a description"></TeaserCard>' language='javascript'/>
