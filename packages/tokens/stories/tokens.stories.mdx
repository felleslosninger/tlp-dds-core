## Kom i gang

#### CDN

[![](https://data.jsdelivr.com/v1/package/npm/@digdir/ds-tokens/badge)](https://www.jsdelivr.com/package/npm/@digdir/ds-tokens)

Bruk av design tokens gjennom CDN (Content Delivery Network). 

Tjenesten jsDelivr er brukt i eksemplene her. Verktøyet 
[https://www.srihash.org/](https://www.srihash.org/) kan brukes til å generere Subresource Integrity hash `integrity="..."` til riktig versjon av ønsket fil fra tokens-pakken. Dette brukes for å sikre at filen som hentes ikke er endret.

__CSS bundle__

<CdnSnippet absoluteFilepath="packages/css/build/index.css" linkFormat="css"/>

**1. Link stilarked i `<head>` i HTML-dokumentet.**

**2. Bruk variablene etter behov i inkluderte CSS-stilark.**

#### NPM

[![](https://img.shields.io/npm/dm/@digdir/ds-tokens.svg)](https://www.npmjs.com/package/@digdir/ds-tokens)

Bruk av design tokens gjennom NPM (Node Package Manager).

**1. Installer pakken i kommandolinjen ved hjelp av NPM eller Yarn.**

`npm install @digdir/ds-tokens`

eller

`yarn add @digdir/ds-tokens`

**2. Importer og bruk deretter variablene i ønsket format.**

##### Bruk i CSS

<CodeSnippet children='/* Importer */
@import "@digdir/ds-tokens/build/tokens.css";
/* Bruk variablene på følgende format: */
:root {
  --my-text-color: var(--color-text-background-light);
}' language='css'/>

##### Bruk i SCSS

<CodeSnippet children='// Importer
@use "@digdir/ds-tokens/build/tokens.scss" as tokens;
// Bruk variablene på følgende format:
$my-text-color: tokens.$color-text-background-light;' language='scss'/>

##### JavaScript

<CodeSnippet children='// Importer
import tokens from "@digdir/ds-tokens/build/tokens.js"
// Bruk variablene på følgende format:
const myTextColor = tokens.color.text.background.light' language='javascript'/>

### Avstander

Avstand-variablene er angitt i `rem`. Denne verdien er relativ til html-elementet sin `font-size`, 
og tar utgangspunkt i den vanligste standarden der __1rem = 16px__.

`spacing-x` variablene speiler settet med definerte avstander som er tilgjengelig i designverktøyet (Figma). 

`spacing-base-x` er et større sett med avstander til dersom man trenger flere verdier enn det som er definert i designet (for eksempel til å håndtere padding inni en komponent man bygger).
Variablene tar utgangspunkt i inkrement på 0.25rem (4px) fra `spacing-base-1` som starter på verdien 0.25rem, slik at `spacing-base-4` er det samme som `4 * 0.25rem == 4 * 4px == 4 * spacing-base-1`, nemlig 1rem (16px).

### Bruddpunkter

Bruddpunktene tar utgangspunkt i verdiene benyttet i [Bootstrap grid](https://getbootstrap.com/docs/5.1/layout/grid/), og følger de samme t-skjortestørrelsene. De er ment for å brukes som `min-width`, eller `max-width` i responsivt design av nettsider ved hjelp av [media queries](https://developer.mozilla.org/en-US/docs/Web/CSS/Media_Queries/Using_media_queries).

For eksempel vil `@media (min-width: var(--breakpoint-medium)) { ... } ` kunne brukes for å treffe de fleste tablets (og større skjermstørrelser).
