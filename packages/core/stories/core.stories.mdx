# Core components

[![](https://img.shields.io/npm/v/@digdir/ds-core-react?label=@digdir/ds-core-react&color=0051be)](https://www.npmjs.com/package/@digdir/ds-core-react)

På denne siden finner du instrukter til hvordan du kan ta i bruk core komponentene.

## Kom i gang

#### NPM

[![](https://img.shields.io/npm/dm/@digdir/ds-core-react.svg)](https://www.npmjs.com/package/@digdir/ds-core-react)

Bruk av core components gjennom NPM (Node Package Manager).

**1. Installer pakken i kommandolinjen ved hjelp av NPM eller Yarn:**

##### NPM
`npm install @digdir/ds-core-react styled-components`
##### Yarn
`yarn add @digdir/ds-core-react styled-components`

**2. Installer avhengigheter:**

##### NPM
`npm install @digdir/ds-tokens`
##### Yarn
`yarn add @digdir/ds-tokens`

**3.Importer komponentene inn og bruk de på denne måten:**

##### JavaScript / Typescript

<CodeSnippet children='// Importer
import { Button } from "@digdir/ds-core-react"
// Bruk komponenten
<Button variant="secondary">Les mer</Button>' language='javascript'/>
