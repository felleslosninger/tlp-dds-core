# Ikoner
[![](https://img.shields.io/npm/v/@digdir/ds-icons?label=@digdir/ds-icons&color=0051be)](https://www.npmjs.com/package/@digdir/ds-icons)

Digdir designsystem sine ikon baserer seg på feather icons, men med ein del justeringer og nye ikon lagt til.

Disse kan du søke på på sida `Ikonsøk`, som du finn i menyen til venstre.

Ikona ligg i NPM pakken: `@digdir/ds-icons`.
Akkuratt no er det kun ein outline versjon av ikona, men me vil og leggje til ein solid variant ved behov.





**Eigenskapar**

* Format: SVG
* Variant: *outlined*
* Størrelse: 24x24 px

## Kom i gang
[![](https://img.shields.io/npm/dm/@digdir/ds-icons.svg)](https://www.npmjs.com/package/@digdir/ds-icons)

**1. Installer pakken i kommandolinja ved hjelp av NPM eller Yarn**

`npm install @digdir/ds-icons`

eller

`yarn add @digdir/ds-icons`

**2. Importer og bruk**

Ikona fins både som React komponentar og som inline SVG.

## Innhold og navn


Ikonene er hentet fra https://github.com/google/material-design-icons, de samme ikonnavnene og mappenavnene er benyttet her.
Dette er de samme navnene som man finner i søkesiden https://fonts.google.com/icons, bare i formatet *snake_case* (Shopping Cart -> shopping_cart).
Mappestruktur (for enklere navigering) og minifiserte SVG'er (for lavere avtrykk/pakkestørrelse) er hentet fra https://github.com/material-icons/material-icons.
