# Ikoner
[![](https://img.shields.io/npm/v/@digdir/ds-icons?label=@digdir/ds-icons&color=0051be)](https://www.npmjs.com/package/@digdir/ds-icons)

Digdir designsystem sine ikoner baserer seg på feather icons, men med ein del justeringer og nye ikoner lagt til.

Disse kan søkes i `Ikonsøk` som du finn i menyen til venstre.

Ikonene ligger i pakken: `@digdir/ds-icons` på NPM.
Akkuratt no er det kun ein outline versjon av ikonene, men me vil og leggje til ein solid variant ved behov.





**Egenskaper**

* Format: SVG
* Variant: *outlined*
* Størrelse: 24x24 px

## Kom i gang
[![](https://img.shields.io/npm/dm/@digdir/ds-icons.svg)](https://www.npmjs.com/package/@digdir/ds-icons)

**1. Installer pakken i kommandolinjen ved hjelp av NPM eller Yarn**

`npm install @digdir/ds-icons`

eller

`yarn add @digdir/ds-icons`

**2. Importer og bruk**

Ikonene finnes både som React komponentar, og som inline SVG.

React:
```javascript
import { Building } from '@digdir/ds-icons'
```
```html
<Building color="blue" height="32px" width="32px">
```
SVG:
```javascript
import BuildingIcon from '@digdir/ds-icons/svg/outline/Building.svg'
```
